# Washing Machine Controller

ATmega2560 based project which emulates the basic functions of a washing machine.

Functionality

* Motor Rotation / Rotation Speed Control (H Bridge)
* Heater Control  (SCR control)
* Valve / Pump Relay control

Microcontroller

AVR - ATMEGA 2560
* PWM Control
* GPIO

Microcontroller Support

* Reset Button
* Linear Regulator
* Analog / Digital source split

H Bridge
* MOSFET Control with PWM and H Bridge

SCR Control
* Optocoupler and TRIAC

Relay Control

* Mosfet coupled with diode protected relays
